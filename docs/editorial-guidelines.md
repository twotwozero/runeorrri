# 편집 가이드

## 독자

국내 일반 러너를 기준으로 씁니다. 풀코스 기록 향상에 몰입한 상급자만이 아니라, 5K, 10K, 하프, 러닝 크루, 장비 구매, 대회 참가에 관심 있는 사람을 포함합니다.

## 톤

- 친근한 큐레이터처럼 씁니다.
- 문장은 짧고, 판단은 분명하게 합니다.
- “이 소식이 왜 중요한지”를 꼭 붙입니다.
- 과장된 홍보 문구, 브랜드 보도자료 말투, 번역투를 피합니다.

좋은 문장:

- 이번 주말 장거리주를 계획 중이라면 교통 통제와 집결 시간을 먼저 확인하세요.
- 기록보다 완주 경험을 중시하는 입문 러너에게 맞는 이벤트입니다.
- 해외 기록 소식이지만, 카본화 경쟁 흐름을 보는 데 참고할 만합니다.

피할 문장:

- 러닝 문화의 새로운 패러다임을 제시했습니다.
- 독보적인 기술력으로 러너의 퍼포먼스를 극대화합니다.
- 뜨거운 반응을 얻으며 성황리에 마무리되었습니다.

## 선별 기준

우선순위가 높은 소식:

- 접수 마감이 가까운 국내 대회
- 일반 러너가 참가할 수 있는 이벤트
- 실제 구매 판단에 도움이 되는 장비 정보
- 러닝 안전, 건강, 훈련에 직접 관련된 정보
- 한국 러너에게 의미 있는 해외 대회, 기록, 트렌드

우선순위가 낮은 소식:

- 단순 브랜드 홍보
- 출처가 불명확한 커뮤니티 소문
- 원문 확인이 어려운 일정 정보
- 일반 러너와 연결점이 약한 엘리트 기록

## 저작권 및 인용

- 원문을 길게 베끼지 않습니다.
- 직접 인용은 꼭 필요한 짧은 문장만 사용합니다.
- 요약은 직접 새로 씁니다.
- 이미지 사용은 공식 배포 이미지, 직접 제작 카드, 또는 사용 허가가 명확한 자료만 사용합니다.
- 모든 소식에는 원문 링크를 붙입니다.

## 카카오/인스타 변환 기준

카카오 채널:

- 5개 소식 전체를 한 번에 발행
- 각 소식은 3~5줄 이내
- 링크 접근성을 우선

인스타그램:

- 1장: 표지
- 2~6장: 소식 5개
- 마지막 장: 저장/공유 유도 문구
- 카드 안 문장은 2줄 단위로 끊어 읽기 쉽게 작성
